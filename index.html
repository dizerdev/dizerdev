Projetos realizados

Ecommerce Model

Plataforma ecommerce com integração Stripe e Paypal, que utiliza o framework Next.js v15 para contruir um projeto em base Node.js 18+, estilização em Tailwind CSS v4, React.js v19, e diversas outras bibliotecas complementares ao React e Next, como Shadcn/ui, que utiliza sistema de componentes de interface, simples de estilizar e construir um site moderno e responsivo. Atualmente utiliza MongoDB e Mongoose ODM, mas esta sendo migrado a utilizar PostgreSQL e Node Postgres. O projeto também conta com uma interface para usuarios "Admin", que oferece controle de cadastro de produtos, de pedidos, usuarios, de páginas publicas do site e de configurações internas.

Speedometer (Mobile)

Projeto desenvolvido em JavaScript puro, que contém o algoritmo utilizado para medir a exata localização a partir do Geolocation do browser e do Local Storage para armazenar os dados. Integra com o Open Street Map, que oferece componentes de interface para diversos ipos de mapa.

Todo List

Site estático para cadastrar lista de tarefas diárias, com um relógio.
