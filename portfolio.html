Ecommerce Model

Plataforma de ecommerce construída com Next.js v15, React v19 e estilização via Tailwind CSS v4. Integra com Stripe e PayPal para pagamentos online e utiliza bibliotecas como Shadcn/ui para UI moderna e responsiva. Inicialmente baseada em MongoDB com Mongoose, está sendo migrada para PostgreSQL com Node Postgres. Possui painel administrativo para gerenciamento de produtos, pedidos, usuários e conteúdo do site. Foco em arquitetura moderna com tecnologias atualizadas e boa experiência para o usuário final e administradores.

Flask Ecommerce API

API de ecommerce desenvolvida com Flask, estruturada em arquitetura RESTful e escrita em Python 3. Inclui endpoints completos para autenticação, cadastro e gestão de usuários, produtos, pedidos e pagamentos. Integrações com gateways como Stripe e PayPal, além de suporte a tokens JWT e CORS configurado. Utiliza PostgreSQL como banco de dados relacional, com SQLAlchemy para ORM e migrações com Alembic. Foco em escalabilidade, segurança e documentação clara via Swagger/OpenAPI. Se quiser, posso adaptar com detalhes específicos do seu projeto (como autenticação, envio de e-mails, logs, testes, etc.).

Speedometer

Projeto em JavaScript puro que utiliza a API de Geolocalização do navegador para capturar a localização exata do usuário. Os dados são armazenados localmente utilizando o Local Storage, garantindo persistência entre sessões. Integra com o OpenStreetMap para renderização de mapas interativos e componentes visuais customizáveis. Foco em simplicidade, desempenho e independência de frameworks externos. Ideal para aplicações que demandam rastreamento geográfico leve e responsivo diretamente no navegador.
